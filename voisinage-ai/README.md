# voisinage114

Application Flutter Voisin’Âge.

## Fonctionnalités
- Authentification (écrans démo)
- Profil utilisateur
- Rencontres de voisinage
- Choix du rayon (carte en aperçu)
- Invitations aux jeux de société
- Tutorats sur plusieurs thèmes
- Rappels des rendez-vous triés par date

## Lancer l’application
1. Ouvrir le projet dans Android Studio ou VS Code.
2. Synchroniser les dépendances Flutter.
3. Lancer sur un émulateur ou un appareil connecté.
